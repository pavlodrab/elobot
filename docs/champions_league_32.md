# Лига Чемпионов на 32 (лига + 2 кубка)

Шаблон под формат, который вы попросили:

> одна большая лига, играется только первый круг → потом игроки делятся на 2 кубка (Основной и Лига Конфети)

## Что получится в итоге

```
┌─────────────────── Лига (32 игрока, один круг, 31 тур, 496 матчей) ───────────────────┐
│                                                                                       │
│  места 1-8                  места 9-24                          места 25-32           │
│      │                          │                                    │                │
│      │  bye                     │ играют 1/16 (стыки)                │                │
│      └─→ ┐                      └─→ 8 матчей × 2 leg ──→ 8 победителей│                │
│          │                                                  │        │                │
│          └────────── 1/8 (16 игроков) ──→ 1/4 ──→ 1/2 ──→ Финал      │                │
│                       все пары × 2 leg                                │                │
│                         (Основной кубок)                              │                │
│                                                                       ↓                │
│                                                              QF → SF → Final           │
│                                                              (Лига Конфети)            │
└───────────────────────────────────────────────────────────────────────────────────────┘
```

* Все пары в обоих кубках играются **в 2 матча** — проход по сумме голов.
* **Матча за 3-е место нет** (ни в основном кубке, ни в утешительном).
* Сетка кубков расставляется **по итоговому месту в лиге**, а не по глобальному ELO.

## 1. Создать турнир

`/new_tournament` → «📋 Шаблоны» → пролистать список → выбрать
**«Лига Чемпионов (32 — лига + 2 кубка)»**.

Бот сам выставит:
- `groups_only=1` (без авто-плей-оффа на всю лигу),
- 1 группа на 32 игрока,
- один круг (`group_matches_per_pair=1`),
- туры включены, бот сам пересчитает 31 тур,
- `followup_cups_config` для авто-кнопки кубков.

## 2. Заполнить ростер и стартовать

1. `/add_player @username1 @username2 …` (или через панель турнира).
2. Когда 32 игрока зарегистрированы — `/start_tournament <id>`.
3. Бот сделает жеребьёвку и запустит первый тур (если включён ручной режим — выгонит туры по `/next_tour`).

## 3. Играть лигу

Обычный поток: игроки скидывают скрины, OCR/админ подтверждают.
Считается, что лига закончена, когда все 496 матчей со статусом `confirmed`.

## 4. Запустить кубки

В чате, к которому привязан турнир, появится автоматическое сообщение:

```
🏁 Лига Champions League 2026 [VSA] завершена!

📊 Финальная таблица: /standings

🏆 По шаблону «Лига Чемпионов» нужно сделать ещё два кубка:
  • Основной — места 1-24 (сетка с байем для топ-8)
  • Лига Конфети — места 25-32

[ 🏆 Создать кубки (24 + 8) ]
```

**Один тап на кнопку** → бот:
1. Прочитает финальную таблицу лиги.
2. Создаст «Основной кубок» (id будет в ответе бота): 24 игрока, сетка на 32, топ-8 лиги получают bye, места 9-24 играют 1/16 (стыки).
3. Создаст «Лигу Конфети»: 8 игроков с мест 25-32, 8-сетка, никаких баев.
4. Расставит первый раунд по сидам лиги (сид 1 = победитель лиги).

В подтверждении бот покажет ID обоих кубков и количество матчей в первом раунде.

То же самое можно сделать через панель турнира самой лиги: `/tournaments` → выбрать лигу → в самом верху панели будет кнопка «🏆 Создать кубки: топ-24 + утешение 8».

Если что-то не так — есть и текстовая команда:
```
/cl_spawn_cups <id_лиги>
```
Сайз можно переопределить:
```
/cl_spawn_cups <id_лиги> 16 8     # топ-16 в основной, 17-24 в утешительный
```

## 5. После спауна

В карточке лиги кнопка «Создать кубки» **исчезает** (защита от двойного спауна), на её место становятся прямые ссылки:

```
🏆 Основной кубок: id 42      ← открывает панель кубка
🥉 Лига Конфети: id 43
```

Дальше — обычный поток кубка:
- сетку можно посмотреть командой `/bracket <id>`,
- матчи играются как обычно (pending → reported → confirmed),
- автопродвижение по стадиям работает: 1/16 → 1/8 → 1/4 → 1/2 → Финал.

## 6. Что считается тай-брейкером

В лиге — стандартный для бота порядок: **очки → разница голов → забитые → ELO**.

В двухматчевых сериях кубков — **сумма голов за две игры**. Если ничья по сумме голов после двух матчей, бот ставит дополнительные leg'и согласно `playoff_stage_config` (по умолчанию «играем дальше пока не определится победитель»; настраивается через панель плей-оффа).

## Если что-то пошло не так

| Симптом | Что делать |
|---|---|
| Нажал «Создать кубки», бот пишет «Лига ещё не закончена» | Проверь, что все 496 матчей `confirmed`. Если есть зависшие — `/list_pending` или `/walkover`. |
| Бот пишет «follow-up cups already spawned» | Кубки уже созданы (см. кнопки «Основной кубок» / «Лига Конфети» в карточке лиги). Если их случайно создали неправильно — `/finish_tournament` дубли и спросить меня про reset. |
| Нет 32 игроков в ростере | Бот откажется делать спаун (`league has N players, need 32`). Добавь недостающих или переопредели сайзы: `/cl_spawn_cups <id> N M` где N+M ≤ количеству игроков. |
| Кнопки в чате нет, но лига уже окончена | Открой панель турнира руками: `/tournaments` → выбери лигу → кнопка будет в самом верху. Авто-сообщение в чат шлётся ОДИН раз — если его прозевали, ничего страшного. |

## Что бот *не* делает (намеренно)

- Не запускает кубки **автоматически**, без подтверждения. Кнопка/команда нужны явно — чтобы был шанс посмотреть таблицу и тай-брейкеры до жеребьёвки.
- Не привязывает кубки к лиге FK-связью — только текстовый бэклинк `followup_cups_tids='<main>:<cons>'`. Хочешь жёсткой связи и каскадного финиша — открой issue, добавим колонкой и сделаем `/finish_league_chain`.
- Не делает матч за 3-е место в утешительном кубке. Если передумаешь — `/set_third_place <cons_id> on`.
